# 📘 **Chapter 2.4: Loss (Error) Functions**

This chapter explains **how a neural network knows it is wrong**.

👉 Without loss functions, the model cannot learn.

---

## 🔹 1. What is a Loss Function?

### Definition

👉 A **loss function** measures the difference between:

* **Actual value (true answer)**
* **Predicted value (model output)**

---

### Simple Example

* Actual price = ₹100
* Predicted price = ₹80

👉 Error = 20

Loss function tells:

> “How bad is this mistake?”

---

## 🔹 2. Why Loss Function is Important

A neural network learns by:

1. Making predictions
2. Measuring error (loss)
3. Reducing that error

---

### Analogy

Think of learning archery 🎯

* Arrow hits far from target → high loss
* Arrow hits near center → low loss

👉 Goal = minimize error

---

## 🔹 3. Types of Loss Functions

Different problems → different loss functions

---

## 🔸 3.1 Mean Squared Error (MSE)

### Formula

MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2

---

### Use Case

👉 Regression problems (continuous values)

---

### Example

| Actual | Predicted | Error | Squared |
| ------ | --------- | ----- | ------- |
| 100    | 80        | 20    | 400     |

---

### Why Square?

* Removes negative signs
* Penalizes large errors more

---

### Intuition

👉 Big mistakes become **very costly**

---

## 🔸 3.2 Mean Absolute Error (MAE)

### Formula

MAE = \frac{1}{n} \sum_{i=1}^{n} |y_i - \hat{y}_i|

---

### Use Case

* Regression

---

### Difference from MSE

* No squaring
* Less sensitive to outliers

---

### Intuition

👉 Treats all errors equally

---

## 🔸 3.3 Binary Cross-Entropy (Log Loss)

### Use Case

👉 Binary classification (Yes/No)

---

### Formula

L = -\left(y \log(\hat{y}) + (1-y) \log(1-\hat{y})\right)

---

### Example

| Actual | Predicted | Loss |
| ------ | --------- | ---- |
| 1      | 0.9       | Low  |
| 1      | 0.2       | High |

---

### Intuition

👉 Punishes **confident wrong predictions heavily**

---

## 🔸 3.4 Categorical Cross-Entropy

### Use Case

👉 Multi-class classification

---

### Example

```id="9p9tzx"
Actual: Cat
Predicted: [Cat: 0.7, Dog: 0.2, Bird: 0.1]
```

👉 Loss depends on probability of correct class

---

## 🔹 4. Choosing the Right Loss Function

| Problem Type               | Loss Function             |
| -------------------------- | ------------------------- |
| Regression                 | MSE / MAE                 |
| Binary Classification      | Binary Cross-Entropy      |
| Multi-class Classification | Categorical Cross-Entropy |

---

## 🔹 5. How Loss Guides Learning

### Flow

1. Model predicts
2. Loss calculates error
3. Optimization adjusts weights
4. Loss decreases

---

👉 Loss is like a **feedback signal**

---

## 🔹 7. Key Differences (MSE vs Cross-Entropy)

| Feature     | MSE                    | Cross-Entropy              |
| ----------- | ---------------------- | -------------------------- |
| Use         | Regression             | Classification             |
| Behavior    | Penalizes large errors | Penalizes wrong confidence |
| Output type | Continuous             | Probabilities              |

---

## 🔹 8. Common Mistakes

### ❌ Using wrong loss function

✔️ Always match with problem type

---

### ❌ Ignoring scale of data

✔️ Normalize data when needed

---

### ❌ Thinking loss = accuracy

✔️ Loss measures error, not correctness

---

## 🔹 9. Important Insight

👉 Lower loss = better model
👉 But:

* Very low loss on training ≠ good generalization

---

## 🎯 Summary (In One Line)

👉 Loss functions measure how wrong a model is and guide it to improve by reducing error.
