# 📘 **Chapter 7.1: Popular Libraries for Neural Networks**

This chapter moves from theory → to practice.

👉 We now learn **which tools are used to build neural networks in real life**

---

## 🔹 1. Why Do We Need Libraries?

Building neural networks from scratch:

* Complex math
* Time-consuming
* Error-prone

---

👉 Libraries provide:

* Pre-built functions
* Efficient computation
* Easy model building

---

### Simple Analogy

Think of cooking:

* From scratch → very slow
* Using ready tools → fast and efficient

👉 Libraries = **ready-made tools**

---

## 🔹 2. Most Popular Libraries

---

## 🔸 2.1 NumPy

### What It Is

👉 Basic library for numerical computation

---

### Why Important

* Handles arrays (vectors, matrices)
* Used in almost all ML libraries

---

### Example

```python
import numpy as np

x = np.array([1, 2, 3])
```

---

👉 Foundation of all neural network computations

---

## 🔸 2.2 TensorFlow

### What It Is

👉 A powerful deep learning framework developed by Google

---

### Features

* High performance
* GPU/TPU support
* Production-ready

---

### Use Cases

* Large-scale models
* Deployment

---

---

## 🔸 2.3 Keras

### What It Is

👉 High-level API built on TensorFlow

---

### Why Useful

* Simple and beginner-friendly
* Less code

---

### Example

```python
from tensorflow import keras

model = keras.Sequential()
```

---

👉 Best for beginners

---

## 🔸 2.4 PyTorch

### What It Is

👉 A flexible deep learning framework by Meta

---

### Features

* Dynamic computation graph
* Easy debugging
* Research-friendly

---

### Example

```python
import torch

x = torch.tensor([1.0, 2.0])
```

---

👉 Most popular in research today

---

## 🔸 2.5 Scikit-learn

### What It Is

👉 Machine learning library (not deep learning focused)

---

### Use Cases

* Data preprocessing
* Simple models
* Evaluation

---

👉 Often used with neural networks

---

## 🔹 3. TensorFlow vs PyTorch

| Feature      | TensorFlow | PyTorch   |
| ------------ | ---------- | --------- |
| Ease of Use  | Moderate   | Easy      |
| Flexibility  | Medium     | High      |
| Industry Use | High       | Growing   |
| Research Use | Moderate   | Very High |

---

👉 Today:

* PyTorch → research
* TensorFlow → production
